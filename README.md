# Alura Store Latam — Análisis de Datos con Python y Pandas

Este repositorio contiene el trabajo realizado para el **challenge de Alura Latam** sobre análisis de datos de una tienda (*Alura Store*), utilizando **Python**, **Pandas**, **NumPy** y otras herramientas de análisis en *dataframes*.  
El proyecto se desarrolló en un **Google Colab Notebook** y tiene como objetivo explorar, limpiar y analizar un conjunto de datos de ventas para extraer *insights* y responder preguntas relevantes de negocio.

## Descripción del Proyecto

El proyecto consiste en realizar un análisis de datos de ventas de **Alura Store** para comprender el comportamiento de clientes y productos, identificar patrones de compra y responder preguntas clave como:

- ¿Cuáles son las principales categorías de productos?
- ¿Qué tendencias de ventas se observan por mes o por tienda?
- ¿Cuáles son los principales KPI (indicadores clave) que aporta el dataset?

El análisis se realiza principalmente con **Pandas**, empleando operaciones de limpieza, organización y agregación de datos para extraer conclusiones valiosas.

---

## Objetivos

- Cargar y explorar el dataset de ventas  
- Limpiar valores faltantes o inconsistentes  
- Realizar análisis exploratorio de datos (EDA)  
- Generar visualizaciones que ayuden a interpretar los datos  
- Responder preguntas de negocio relevantes

## Datos Utilizados

Los datos se encuentran en el notebook principal **`AluraStoreLatam.ipynb`** y consisten en registros de ventas de la tienda ficticia **Alura Store**.

El dataset incluye:

- Identificadores de venta, producto y tienda  
- Categoría, cantidad y montos de venta  
- Fechas de compra  
- Otras columnas relevantes para el análisis
